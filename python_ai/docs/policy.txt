=== 회사 지출 규정 ===

1. 식비 규정
   - 1인당 식비: 최대 5만원
   - 회식비: 최대 20만원 (4인 기준)
   - 2만원 이상: 영수증 필수
   - 2만원 미만: 영수증 선택 가능 (간단한 식사, 커피 등)
   - 개인 용도 식비는 반려

2. 교통비 규정
   - 택시비: 5만원 이하는 영수증 선택 가능
   - 5만원 초과: 영수증 필수
   - 대중교통: 영수증 불필요 (교통카드 사용 내역으로 대체 가능)
   - 개인 용도 교통비는 반려

3. 비품 구매 규정
   - 5만원 이하: 영수증 선택 가능 (소액 사무용품 등)
   - 5만원 초과: 영수증 필수
   - 모든 비품 구매는 관리자 승인 필요
   - 업무 관련 비품만 승인

4. 반려 기준
   - 금액 불일치: 즉시 반려 (REJECT_CLEAR)
   - 날짜 불일치: 즉시 반려 (REJECT_CLEAR)
   - 영수증 없이 10만원 이상: 반려 (REJECT_SUSPECTED)
   - 영수증 없이 5만원 이상 10만원 미만: 추가 검토 필요 (REJECT_SUSPECTED)
   - 개인 용도 의심: 반려 (REJECT_CLEAR)
   - OCR 신뢰도 70% 미만: 추가 검토 필요 (REJECT_SUSPECTED)

5. 승인 기준
   - 모든 정보 일치
   - 영수증 신뢰도 70% 이상
   - 업무 관련성 명확
   - 규정 준수
   - 소액 지출(2만원 미만)은 영수증 없이도 승인 가능
